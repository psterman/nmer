## 增强内容

### 2. **检测优先级**
按以下顺序检测：
1. 文件/文件夹路径（优先，因为路径可能包含其他字符）
2. 链接
3. 邮箱
4. 手机号
5. 代码
6. 文本（默认）

### 3. **统计信息**
自动计算并保存：
- 字符数 (CharCount) - 去除首尾空白后的字符数
- 字数 (WordCount) - 去除标点后按空格分割的词数

### 4. **数据保存**
CapsLock+C 复制时：
- 自动调用 `ClassifyClipboardContent` 进行类型分类
- 自动计算字符数和字数
- 保存到数据库时包含所有分类和统计信息

当用户使用 `CapsLock+C` 复制内容时：

1. 捕获内容 → 获取剪贴板内容
2. 环境捕获 → 获取来源应用、窗口标题、程序路径
3. 智能分类 → 调用 `ClassifyClipboardContent` 进行类型分类
4. 统计计算 → 自动计算字符数和字数
5. 保存到数据库 → 包含所有信息（类型、统计、来源等）
