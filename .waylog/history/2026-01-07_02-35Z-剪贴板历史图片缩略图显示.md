# 剪贴板历史图片缩略图显示
_Exported on 01/07/2026 at 10:38:13 GMT+8 from Cursor via WayLog_

我需要在剪贴板增加一个复制图片的缩略图显示功能：以下是实
1. 核心原理
监听剪贴板：当 A_Clipboard 变化且类型为图片时触发。

提取并缩放：使用 GDI+ 获取剪贴板图片，将其按比例缩放到小尺寸。

缓存存储：将缩略图保存到本地磁盘（如 Cache\Thumbs），或直接加载到内存中的 ImageList。

UI 绑定：在 ListView 中通过 Icon 索引关联该图片。

2. 修改 ClipboardHistoryPanel.ahk 实现缩略图显示
第一步：初始化 ImageList (在 InitClipboardHistoryPanel 中)
ListView 必须关联一个 ImageList 才能显示图标。

AutoHotkey

; 在 InitClipboardHistoryPanel 函数内，创建 ListView 后添加：
global HistoryImageList := IL_Create(10, 10, 0) ; 初始创建 10 个槽位，允许自动增长
HistoryListView.SetImageList(HistoryImageList, 1) ; 1 表示小图标模式
第二步：处理图片并加入列表 (在 RefreshHistoryList 中)
修改循环逻辑，当发现 DataType = "Image" 时，加载缩略图。

AutoHotkey

RefreshHistoryList(SearchText := "") {
    global HistoryImageList, HistoryImageListCache
    HistoryListView.Delete()
    
    ; 清空旧的图标缓存（可选，视性能而定）
    ; IL_Destroy(HistoryImageList)
    ; HistoryImageList := IL_Create(10, 10, 0)
    ; HistoryListView.SetImageList(HistoryImageList, 1)

    for row in HistoryDisplayCache {
        iconIndex := -1 ; 默认无图标
        
        if (row.Has("DataType") && row["DataType"] = "Image") {
            imagePath := row["Content"]
            
            ; 检查缓存避免重复加载
            if (HistoryImageListCache.Has(imagePath)) {
                iconIndex := HistoryImageListCache[imagePath]
            } else {
                if FileExist(imagePath) {
                    ; 直接将图片文件路径加入 ImageList，系统会自动处理简单的缩放
                    iconIndex := IL_Add(HistoryImageList, imagePath)
                    HistoryImageListCache[imagePath] := iconIndex
                }
            }
            displayContent := "[图片] " . row["SourceApp"]
        } else {
            iconIndex := 0 ; 文字记录不显示图标，或显示默认图标
            displayContent := StrReplace(row["Content"], "`n", " ")
        }
        
        ; 使用 Icon 选项添加行
        HistoryListView.Add("Icon" . iconIndex, row["Timestamp"], displayContent)
    }
}
3. 在复制时“预处理”缩略图 (优化性能)

在你的主脚本（如 CursorHelper.ahk）的剪贴板监听逻辑中：

AutoHotkey

OnClipboardChange(Type) {
    if (Type = 2) { ; 剪贴板是图片
        try {
            ; 1. 保存原图
            CacheDir := A_ScriptDir "\Cache\Images"
            ThumbDir := A_ScriptDir "\Cache\Thumbs"
            if !DirExist(ThumbDir)
                DirCreate(ThumbDir)
                
            FileName := A_Now ".png"
            FullScalePath := CacheDir "\" FileName
            ThumbPath := ThumbDir "\" FileName
            
            ; 使用 ImagePut 保存原图
            ImagePut.Save(ClipboardAll(), FullScalePath)
            
            ; 2. 【核心】使用 Gdip 生成缩略图 (例如 64x64)
            pToken := Gdip_Startup()
            pBitmap := Gdip_CreateBitmapFromFile(FullScalePath)
            ; 保持比例缩放至 64 像素宽
            pThumb := Gdip_ResizeBitmap(pBitmap, 64, 64, true) 
            Gdip_SaveBitmapToFile(pThumb, ThumbPath)
            
            Gdip_DisposeImage(pBitmap)
            Gdip_DisposeImage(pThumb)
            Gdip_Shutdown(pToken)
            
            ; 3. 存入数据库时记录 ThumbPath
            ; ... 你的 SQL 插入逻辑 ...
        }
    }
}
4. 关键点提示
Gdip_ResizeBitmap: 确保你的 lib\Gdip_All.ahk 中包含此函数，如果没有，可以使用 Gdip_DrawImageRectI 来实现缩放绘图。

内存管理: HistoryImageListCache 是一个 Map。如果用户复制了上千张图片，这个 Map 会占用一定内存，建议在面板关闭或定期进行清理。

显示模式: ListView 的图标显示效果受 View 模式影响。

Report 模式：图标较小（16x16）。

Icon 或 Tile 模式：可以显示较大的预览图，更适合“图片管理”面板。
