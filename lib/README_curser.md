# Curser - Cursor Chat Exporter

Cursor 编辑器聊天记录导出工具，一款专为程序员打造的AI对话管理神器！

## ✨ 核心特性

### 🎨 界面设计
- **现代化GUI**：基于AHK开发的原生Windows界面
- **主题切换**：支持暗色/亮色主题，参考v2ex设计风格
- **响应式布局**：适配不同屏幕尺寸和分辨率
- **直观操作**：拖拽、双击、右键菜单等丰富交互

### 📊 数据处理
- **智能解析**：自动识别和解析Cursor的SQLite数据库
- **兼容性强**：支持新旧版本Cursor的数据格式
- **高性能**：SQLite原生访问，读取速度飞快
- **错误处理**：完整的异常处理和错误恢复机制

### 🔍 搜索过滤
- **实时搜索**：输入即时反馈，快速定位聊天内容
- **多维度过滤**：按聊天类型、时间范围、工作区过滤
- **智能匹配**：支持关键词、代码片段、正则表达式
- **结果排序**：按相关度、时间等灵活排序

### 📁 导出功能
- **Markdown格式**：带语法高亮的代码块，适合文档编写
- **HTML格式**：美观的网页版本，可分享给他人查看
- **JSON格式**：完整的原始数据，便于程序处理
- **批量导出**：支持选择多个工作区一次性导出

### ⚙️ 高级特性
- **导入恢复**：可以导入之前导出的聊天记录
- **性能优化**：异步处理，避免界面冻结
- **日志记录**：详细的操作日志，方便调试
- **配置持久化**：记住用户的偏好设置

## 🚀 快速开始

### 系统要求
- **操作系统**：Windows 10/11
- **AutoHotkey**：v2.0 或更高版本
- **Cursor编辑器**：已安装并使用过

### 安装步骤

1. **下载文件**
   ```bash
   # 确保所有依赖库都在 lib 目录下
   lib/
   ├── curser.ahk          # 主程序
   ├── Class_SQLiteDB.ahk  # SQLite数据库库
   ├── Gdip_All.ahk        # 图形处理库
   └── 其他依赖文件...
   ```

2. **运行程序**
   ```bash
   # 双击运行 curser.ahk 文件
   curser.ahk
   ```

3. **初始化设置**
   - 程序首次运行会自动创建配置文件
   - 自动扫描Cursor的工作区
   - 设置导出目录和偏好格式

### 基础使用

1. **启动程序**：运行 `curser.ahk`
2. **扫描工作区**：点击"刷新"按钮扫描所有工作区
3. **查看聊天**：双击工作区查看聊天记录
4. **导出数据**：选择聊天后点击"导出聊天"按钮

## 📖 详细功能说明

### 工作区管理

- **自动检测**：程序启动时自动扫描所有Cursor工作区
- **手动刷新**：点击"刷新"按钮重新扫描
- **工作区信息**：显示项目名称、ID、聊天数量、状态
- **批量选择**：支持全选/取消选择多个工作区

### 聊天记录查看

- **列表显示**：以表格形式展示所有聊天
- **详细信息**：标题、类型、消息数量、时间戳
- **双击查看**：双击聊天查看详细内容
- **搜索过滤**：支持实时搜索和类型过滤

### 导出选项

#### Markdown 格式
```markdown

聊天数量: 5

### 👤 用户:
请帮我优化这段代码...

### 🤖 AI:
以下是优化后的代码：
```javascript
// 优化后的代码
function example() {
    // 实现逻辑
}
```
```

#### HTML 格式
- 现代化的网页界面
- 支持暗色/亮色主题
- 代码语法高亮
- 响应式设计，适配手机查看

### 搜索和过滤

- **关键词搜索**：在聊天标题和内容中搜索
- **类型过滤**：只显示聊天或Composer记录
- **时间范围**：按时间段过滤聊天记录
- **工作区过滤**：只查看特定项目的聊天

### 配置选项

程序会在 `config/curser.ini` 中保存用户配置：

```ini
[Settings]
Theme=dark
ExportFormat=markdown
ExportDir=C:\Users\用户名\Desktop\小c\output
SearchDebounce=300
MaxPreviewLines=100
```

## 🛠️ 技术架构

### 核心模块

```
curser.ahk (主程序)
├── CurserApp 类          # 主应用逻辑
├── GUI 界面模块          # 用户界面
├── 数据库解析模块        # SQLite数据处理
├── 导出引擎模块          # 多格式导出
├── 搜索过滤模块          # 内容搜索
└── 配置管理模块          # 设置持久化
```

### 依赖库

- **Class_SQLiteDB.ahk**：SQLite数据库访问
- **Gdip_All.ahk**：图形和图像处理
- **JSON处理模块**：内置的JSON解析和序列化

### 数据流程

1. **扫描阶段**：检测Cursor工作区和数据库文件
2. **解析阶段**：读取SQLite数据库，解析聊天数据
3. **处理阶段**：应用搜索过滤，准备导出数据
4. **导出阶段**：转换为目标格式，写入文件

## 🔧 故障排除

### 常见问题

**Q: 程序启动失败**
A: 确保所有依赖库文件都在 lib 目录中

**Q: 找不到工作区**
A: 确认Cursor已安装并创建过工作区

**Q: 导出失败**
A: 检查输出目录权限和磁盘空间

**Q: 搜索无结果**
A: 尝试简化关键词或检查过滤条件

### 日志调试

程序会在 `logs/curser.log` 中记录详细日志：

```
[2024-01-10 15:30:00] 程序启动
[2024-01-10 15:30:01] 找到 4 个工作区
[2024-01-10 15:30:02] 解析工作区: abc123
[2024-01-10 15:30:03] 找到 15 个聊天记录
```

### 性能优化

- **异步处理**：导出大文件时不会冻结界面
- **内存管理**：及时释放不再使用的数据
- **缓存机制**：避免重复解析相同数据
- **分页加载**：大量聊天记录时分批显示
