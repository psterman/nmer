# CapsLock+F 大写问题 - 完整修复指南

## 问题
按下 CapsLock+F 进入搜索中心后，CapsLock 大写灯亮起

## 根本原因
```
用户按下 CapsLock
    ↓
~CapsLock:: 触发（~ 表示系统也处理）
    ↓
系统立即切换大写 ON（灯亮）
    ↓
用户按 F
    ↓
虽然脚本尝试关闭，但竞争条件下大写保持 ON
```

## 解决方案：完全禁用 CapsLock 大写功能

### 方法一：一键自动修复

**双击运行**：`Fix_CapsLock_Now.ahk`

自动完成以下修改：
1. 添加 `#InstallKeybdHook`
2. 添加 `SetCapsLockState("AlwaysOff")`
3. 将 `~CapsLock::` 改为 `CapsLock::`
4. 在 `f::` 热键中添加强制关闭
5. 在 `ShowSearchCenter()` 中添加强制关闭

**然后**：
1. 完全退出脚本（托盘图标右键 -> 退出）
2. 重新启动脚本
3. 测试 CapsLock+F

### 方法二：手动修复（详细步骤）

#### 步骤1：脚本开头添加禁用代码

**位置**：第 7 行（基础配置之后）

**添加**：
```ahk
#InstallKeybdHook
SetCapsLockState("AlwaysOff")
Loop 5 {
    SetCapsLockState("AlwaysOff")
    Sleep(50)
}
```

#### 步骤2：修改 ~CapsLock:: 热键

**位置**：约第 3799 行

**修改前**：
```ahk
~CapsLock:: {
```

**修改后**：
```ahk
CapsLock:: {
```

**在 KeyWait("CapsLock") 后添加**：
```ahk
KeyWait("CapsLock")
SetCapsLockState("AlwaysOff")  ; 添加这行
```

#### 步骤3：修改 f:: 热键

**位置**：约第 22984 行

**修改前**：
```ahk
f:: {
    global IsCountdownActive
    ; 如果倒计时正在进行，按下 F 立即加速执行（发射内容）
```

**修改后**：
```ahk
f:: {
    global IsCountdownActive
    
    ; 【修复】强制关闭 CapsLock
    SetCapsLockState("AlwaysOff")
    
    ; 如果倒计时正在进行，按下 F 立即加速执行（发射内容）
```

**在结尾添加**：
```ahk
        ShowSearchCenter()
    }
    
    ; 【修复】执行后强制关闭 CapsLock
    SetCapsLockState("AlwaysOff")
}
```

#### 步骤4：修改 ShowSearchCenter 函数

**位置**：约第 23983 行

**修改前**：
```ahk
ShowSearchCenter() {
    global GuiID_SearchCenter, UI_Colors, ThemeMode
```

**修改后**：
```ahk
ShowSearchCenter() {
    ; 【修复】强制关闭 CapsLock
    SetCapsLockState("AlwaysOff")
    
    global GuiID_SearchCenter, UI_Colors, ThemeMode
```

## 修复后效果

| 场景 | 修复前 | 修复后 |
|------|--------|--------|
| CapsLock+F | 灯亮 ❌ | 灯不亮 ✅ |
| 单击 CapsLock | 可能异常 ❌ | 切换输入法 ✅ |
| 其他快捷键 | 可能异常 ❌ | 正常 ✅ |

## 如果输入法无法切换

某些输入法使用 Shift 切换中英文，修改 CapsLock 热键：

```ahk
} else {
    ; 使用 Shift 切换输入法
    Send("{LShift}")
    CapsLock := false
}
```
