# 快捷键演示图片说明

## 图片文件要求

请将 CapsLock + C 的演示图片保存为以下文件名，并放在此目录中：

- **hotkey_c.png** - CapsLock + C（连续复制）功能的演示图片

## 图片规格建议

- 格式：PNG（推荐）或 JPG
- 尺寸：建议宽度 800-1200px，高度根据内容自适应
- 内容：应展示连续复制功能的使用流程和效果

## 其他快捷键图片（可选）

如果需要为其他快捷键添加图片，可以使用以下命名：

- hotkey_v.png - CapsLock + V（合并粘贴）
- hotkey_x.png - CapsLock + X（剪贴板管理）
- hotkey_e.png - CapsLock + E（解释代码）
- hotkey_r.png - CapsLock + R（重构代码）
- hotkey_o.png - CapsLock + O（优化代码）
- hotkey_q.png - CapsLock + Q（配置面板）
- hotkey_z.png - CapsLock + Z（语音输入）
- hotkey_s.png - CapsLock + S（分割代码）
- hotkey_b.png - CapsLock + B（批量操作）
- hotkey_esc.png - ESC（关闭面板）

## 注意事项

- 如果图片文件不存在，设置面板会显示提示文本
- 图片路径相对于脚本文件所在目录
- 建议使用相对路径，便于项目迁移
